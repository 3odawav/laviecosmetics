# **App Name**: DocuSummarizer

## Core Features:

- Document Upload: Allow users to upload documents from various sources such as local storage or via URL.
- AI Summarization: Leverage AI to automatically summarize the uploaded document.
- Summary Display: Present the summarized content in a clear, readable format with options to adjust summary length.
- Document Viewing: Provide an interface to view the original document alongside the summary.
- Keywords Extraction: Use AI as a tool to extract the most important keywords from the document for quick reference.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey trust and efficiency, aligning with the app's purpose.
- Background color: Light gray (#F5F5F5), providing a clean, distraction-free environment for reading.
- Accent color: Muted teal (#4DB6AC), used for interactive elements like buttons and highlights, creating a subtle contrast.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look; readable and professional, well-suited for both summaries and original text.
- Use minimalist icons representing document types and actions (upload, summarize, download) from a free icon library such as Font Awesome or Material Icons.
- Employ a clean, intuitive layout with a clear division between the document viewing area and the summary display. Prioritize readability and ease of navigation.
- Subtle animations to indicate processing and loading states, ensuring a smooth user experience.
